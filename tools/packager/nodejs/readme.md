# wa-kit

The [package `wa-kit`](https://www.npmjs.com/package/wa-kit) provides JavaScript developers with comprehensive definitions extracted from WhatsApp Web, enabling the development of unofficial WhatsApp automations and integrations.

## What's Included

This package extracts and provides multiple types of definitions from WhatsApp Web:

- **Protocol Buffers**: Message schemas and data structures used in WhatsApp communication protocol (compatible with [`@bufbuild/protobuf@1.10.0`](https://www.npmjs.com/package/@bufbuild/protobuf/v/1.10.0))
- **GraphQL Schemas**: Types and definitions from WhatsApp GraphQL schema
- **Binary Schemas**: Low-level binary protocol definitions
- **JID Definitions**: WhatsApp identifier patterns and structures
- **Constants**: Various constants and enums used throughout WhatsApp Web

✨ Definitions were last extracted from **WhatsApp Web version**: `{{WA_VERSION}}`.

**⚠️ This package is automatically updated when changes are detected in WhatsApp Web.**

## Installation

To install the package, use npm:

```sh
npm install wa-kit
```
